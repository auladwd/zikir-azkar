# আযকার — জিকির ও দোয়া অ্যাপ ☪

সকাল-সন্ধ্যা এবং পাঁচ ওয়াক্ত ফরয নামাজের পরের জিকির ও দোয়া পড়ার একটি সুন্দর ও আকর্ষণীয় মোবাইল-ফার্স্ট ওয়েব অ্যাপ।

## 🌟 বৈশিষ্ট্য

- 🌅 **সকালের আযকার** — ১১টি সম্পূর্ণ দোয়া (আয়াতুল কুরসি, সাইয়্যিদুল ইস্তিগফার সহ)
- 🌙 **সন্ধ্যার আযকার** — ৯টি বিশেষ সন্ধ্যার দোয়া
- 🕌 **পাঁচ ওয়াক্ত নামাজের পরের দোয়া** — ফজর, যোহর, আসর, মাগরিব, ইশা
- 📿 **ডিজিটাল তাসবিহ কাউন্টার** — কাস্টম জিকির সহ
- ❤️ **প্রিয় দোয়া** — পছন্দের দোয়া সংরক্ষণ
- 📊 **আমলের ইতিহাস** — দৈনিক প্রগতি ও সাপ্তাহিক চার্ট
- 🎨 **তিনটি থিম** — ডার্ক, লাইট, নেচার
- 📡 **অফলাইন সাপোর্ট** — ইন্টারনেট ছাড়াও কাজ করে (PWA)
- 🔍 **দোয়া অনুসন্ধান** — বাংলায় সার্চ করুন

## 🚀 ব্যবহার

সরাসরি ব্রাউজারে `index.html` খুলুন অথবা যেকোনো Static Server দিয়ে চালান।

```bash
npx serve . -p 3000
```

তারপর `http://localhost:3000` এ যান।

## 📱 মোবাইলে ইনস্টল করুন

Chrome-এ অ্যাপ খুলে **"Add to Home Screen"** বা **"Install App"** করলে পূর্ণ মোবাইল অ্যাপের মতো কাজ করবে।
